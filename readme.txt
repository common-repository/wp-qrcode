<h4>Description</h4>

This is a free contemporary multi-purpose responsive Plugin will Generate the Qrcode Based on Shortcode


== Installation ==

1. Upload 'wp-Qrcode' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Then use the following shortcode for showing the portfolio. 
  [auto_qrcode qrcode="best" size=6 ,margin=4]   
               or
  [auto_qrcode qrcode="https://sites.google.com/view/pikashow-apk-downloads/" size=6 ,margin=4]
4. Here Qrcode is the Text of Qrcode ,Size is the Qrcode Size and Margin is the Margin of the Qrcode.
5. To Add Qr Code Thought Widgets Go to Widgets and Qr code

= Features =

* Free To Generate QR Code
* Generate Unlimited Qr Code
* Generate Web Url Qr Code
